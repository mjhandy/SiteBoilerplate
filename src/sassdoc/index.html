    gradientAnimation

    @mixin gradientAnimation() { ... }

    Description

    Background Hover Animations This will give the element a gradient background that will animate on hover. You can select the start and stop colours, as well as animation speed and how many times it animates

    Parameters

    None.

    Example

    usage: @include gradientAnimation($blue, darken($blue, 10), bottom, 2s, 2);
